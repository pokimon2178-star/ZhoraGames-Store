# ZhoraGames Store 🎮

**ZhoraGames Store** — это мобильное приложение на языке Kotlin для платформы Android, представляющее собой функциональный магазин видеоигр[cite: 1, 10]. Проект включает в себя полноценную систему авторизации, динамический каталог товаров и механизмы управления личной коллекцией игр через локальную базу данных[cite: 11, 21].

## 🚀 Основные возможности

*   **Система аккаунтов**: Регистрация и вход пользователей с сохранением сессии в `SharedPreferences`[cite: 18, 21].
*   **Динамический каталог**: Список игр (например, GTA 6, Hitman, Elden Ring) загружается напрямую из SQLite[cite: 11, 15].
*   **Интерактивная корзина**: Возможность добавлять товары и оформлять покупку, переводя игры в личную библиотеку[cite: 20].
*   **Избранное**: Сохранение понравившихся игр «под звездочкой» для быстрого доступа[cite: 12, 16].
*   **Личная библиотека**: Список приобретенных игр с возможностью их «запуска» прямо из приложения[cite: 16, 17].

## 🛠 Технический стек

*   **Язык разработки**: Kotlin[cite: 10, 16].
*   **База данных**: SQLite (через `SQLiteOpenHelper`) — реализовано 5 связанных таблиц: `users`, `games`, `cart`, `favorites`, `library`[cite: 11].
*   **UI-компоненты**:
    *   `RecyclerView` с универсальным адаптером `ItemsAdapter` для отображения списков[cite: 15, 16].
    *   `CardView` для современного оформления карточек товаров[cite: 9].
    *   `ConstraintLayout` и `LinearLayout` для адаптивной верстки экранов[cite: 1, 2].
*   **Навигация**: Использование `Intent` для передачи данных между экранами (название игры, цена, описание)[cite: 10, 14, 16].

## 📂 Структура проекта

### Основная логика:
*   `DBHelper.kt` — ядро приложения: управление SQL-запросами, создание таблиц и хранение данных[cite: 11].
*   `ItemsAdapter.kt` — мост между данными и интерфейсом, меняющий логику кнопок в зависимости от контекста (Магазин/Корзина/Библиотека)[cite: 16].
*   `AuthActivity.kt` & `MainActivity.kt` — компоненты для авторизации и регистрации новых пользователей[cite: 18, 21].

### Оформление (XML):
*   `activity_main.xml` — главный экран магазина «ЖОРА ГЕЙМС»[cite: 1].
*   `item_in_list.xml` — шаблон элемента списка с изображением и кнопкой подробностей[cite: 9].
*   `activity_content.xml` — разметка страницы с полным описанием игры[cite: 3].
